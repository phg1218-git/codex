# Page Route Map

- `/` home/entry
- `/(marketing)` marketing placeholder
- `/complete-profile` forced completion page
- `/api/auth/[...nextauth]` auth endpoints
- `/api/profile` profile CRUD endpoint
- `/api/survey` survey definition + submit
- `/api/matching` matchmaking ranking endpoint
- `/api/chat/threads` chat thread endpoint
- `/api/chat/messages` message polling endpoint
